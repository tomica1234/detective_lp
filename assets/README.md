# assets ディレクトリ運用ルール

このディレクトリの画像は、UUID名や `ChatGPT Image ...` のような一時名ではなく、
**用途が分かる名前**に統一してください。

## 命名一覧

- `couple_illustration.png`
- `leaf_decor_right.png`
- `brand_logo_heart.png`
- `camera_illustration.png`
- `night_couple_silhouette.png`
- `checklist_magnifier.png`
- `woman_worried.png`
- `plan_icons_set.png`
- `flow_icons_set.png`

## 旧名から新名へのリネーム例

```bash
mv "8d453eae-4a04-46bc-b7eb-eda262a13692.png" "couple_illustration.png"
mv "20a02e21-e022-41e4-90d4-999f59da8d7f.png" "leaf_decor_right.png"
mv "6282a828-7200-428f-9b47-fa7e1138b1f1.png" "brand_logo_heart.png"
mv "2dfca03e-8927-496d-8403-e379c981962c.png" "camera_illustration.png"
mv "74efa979-a984-4a7d-acf2-6b8becbb0026.png" "night_couple_silhouette.png"
mv "c728279d-c12c-444c-8472-37587239b4b5.png" "checklist_magnifier.png"
mv "d110cba0-d983-4349-a730-a9a21c29d859.png" "woman_worried.png"
mv "ChatGPT Image 2026年4月29日 00_34_03.png" "plan_icons_set.png"
mv "ChatGPT Image 2026年4月29日 00_34_23.png" "flow_icons_set.png"
```
