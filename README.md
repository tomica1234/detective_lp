# 画像配置先

`assets/` 配下の画像は、用途が分かる名前で管理してください。

## 推奨ファイル名（新）

- `couple_illustration.png`（寄り添うカップル）
- `leaf_decor_right.png`（右下の葉飾り）
- `brand_logo_heart.png`（サービスロゴ）
- `camera_illustration.png`（カメラ）
- `night_couple_silhouette.png`（夜景のカップルシルエット）
- `checklist_magnifier.png`（チェックリスト＋虫眼鏡）
- `woman_worried.png`（不安な女性）
- `plan_icons_set.png`（激安プランの各アイコン）
- `flow_icons_set.png`（依頼フローの各アイコン）
