# Codexに渡す実装指示

添付画像 `lp_mock.png` をデザイン参考として、実際に公開できるLPを実装してください。

## 目的
若い未婚カップル向けに、初めて探偵に相談する人が安心して問い合わせできるLPを作る。
「安さ」「相談しやすさ」「お試し感」を強く出す。

## ターゲット
- 10代後半〜20代の未婚カップル
- 「浮気かも」と不安だが、大げさな探偵依頼には抵抗がある人
- 高額契約が怖い人
- まずは軽く確認したい人

## メイン訴求
- 初回限定 激安プラン
- 5,000円 / 時間
- 1回のみ
- 最大3時間
- 別途諸経費
- 未婚・未婚約者限定
- 写真と事実のみ報告
- 詳細レポートなし

## 補助プラン
### カップルプラン
- 3万円 / 5時間
- しっかり確認したい方向け
- 基本単位は1時間
- 詳細レポート追加 +4万円

## LP構成
1. ヘッダー
   - ロゴ
   - 電話番号/相談導線
2. ファーストビュー
   - 初めての探偵相談でも安心
   - 未婚カップル向けのやさしい調査サービス
   - 不安な気持ちに寄り添い、丁寧にサポート
   - 5,000円 / 時間を大きく表示
3. 悩み共感
   - 恋人の行動が気になる
   - 信じたいけど不安を確認したい
   - 二人の将来のために事実を知りたい
4. 激安プラン詳細
   - 未婚・未婚約者限定
   - 最大3時間まで
   - 指定場所で張り込み
   - 数枚の写真と事実のみ報告
   - 詳細レポートなし
5. 選ばれる理由
   - はじめてでも相談しやすい
   - 秘密厳守で安心
   - 明確な料金で安心
6. カップルプラン
7. ご依頼の流れ
8. FAQ
9. 最終CTA

## デザイン要件
- 添付画像の雰囲気を参考にする
- 白〜薄い水色ベース
- メインカラーはピンク、アクセントに青
- 角丸のカードを多用
- スマホファースト
- 料金を大きく見せる
- CTAボタンはピンクのグラデーション
- 怖い探偵感、黒背景、高圧的表現は避ける
- 「浮気を暴く」より「不安確認」「事実確認」のトーン

## 実装要件
- 既存プロジェクト構成に合わせる
- 既存CSS/Tailwindがあれば再利用
- 画像内の文字を完全コピーする必要はない
- h1/h2構造を整理
- スマホ・タブレット・PC対応
- CTAは複数箇所に配置
- アクセシビリティとしてalt、button、linkを整える

## 検証
Playwrightまたはブラウザで以下の幅を確認し、崩れがあれば修正してください。
- 375px
- 768px
- 1440px

確認項目:
- スマホのファーストビューで料金とCTAが見える
- 料金カードが崩れない
- CTAが押しやすい
- FAQが読みやすい
- 添付画像の雰囲気から大きく外れていない
